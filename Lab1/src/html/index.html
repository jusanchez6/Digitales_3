Documentación del Proyecto de Cifrado AES

Introducción

Como primer laboratorio de la asignatura de Electronica Digital 3, se propone la implementación de un cifrado AES-128 en C. El cifrado AES (Advanced Encryption Standard) es un algoritmo de cifrado simétrico que se utiliza para proteger datos confidenciales. AES-128 es la variante más común de este algoritmo y utiliza una clave de 128 bits para cifrar los datos. esto para el entendimiento de operaciones a nivel de bit y el acecamiento a la programación en lenguaje C.

Este proyecto implementa el algoritmo de cifrado AES-128 en C. El archivo principal, main.c, contiene la función principal que gestiona la encriptación de datos. A continuación se detallan los componentes principales y el flujo de ejecución del programa.

Detalles del Programa

  • La clave y el estado se leen desde archivos de texto.
  • La clave se expande para el cifrado AES-128.
  • El estado se encripta utilizando la clave expandida.
  • El resultado se guarda en el archivo encripted.txt.
